# Deploy Unmotivated Hero to Vercel (Free)

## Steps to access from your iPad:

1. Go to https://vercel.com and sign up (free)
2. Click "Add New Project"
3. Import your git repository: `git-test`
4. Set Root Directory to: `frontend`
5. Add Environment Variable:
   - Name: `NEXT_PUBLIC_API_URL`
   - Value: `http://YOUR_BACKEND_URL/api/v1` (or use localhost for now)
6. Click "Deploy"
7. You'll get a URL like: `https://your-app.vercel.app`
8. Access that URL from your iPad!

The deployment is free and takes about 2 minutes.
